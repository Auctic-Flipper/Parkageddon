Real-time visibility for campus parking

Parkageddon aggregates and visualizes garage capacity to help students, faculty, and visitors find parking quickly.

What is this?

This dashboard shows the approximate number of available car and motorcycle spaces by floor for each garage on campus. It’s an early version and will continue to improve throughout the semester.

Data sources

Data is seeded for development and can be updated programmatically. Historical changes are stored for analysis over time.
